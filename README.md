# ☕ Coffee Rating App

A full-stack coffee rating application where users can view coffee items and vote for their favorites. Votes are stored permanently using a backend API and SQLite database.

##  Features

- View coffee list with images and descriptions
- Vote for your favorite coffee
- Real-time vote updates
- Leaderboard sorted by votes
- Votes stored permanently in database
- Clean and responsive user interface

## Tech Stack

### Frontend
- React
- Vite
- Axios
- CSS

### Backend
- Flask
- Python
- SQLite
- REST API
